# 🌐 Unicloud
**One Place for All Your Cloud Storage**

Unicloud is a unified cloud platform that brings multiple storage services into a single, seamless interface. Instead of switching between different apps, logging in and out, and struggling with storage limits, users can access, view, manage, and upload their files and photos from one central, beautifully designed dashboard.

Built to simplify digital storage, Unicloud focuses on accessibility, clarity, and efficiency—especially for users who rely on multiple cloud platforms daily.

URL: https://unicloud-services.netlify.app/

---

## 🚀 What It Does

Unicloud connects your cloud accounts (like multiple Google Drive accounts) and presents all your data in one place.

- **Unified Access:** Access files from multiple cloud services without switching apps.
- **Smart Uploads:** Drop massive files (up to 5GB) into the platform, and Unicloud will automatically scan your accounts to find the one with the most free space to upload it to.
- **Infinite Gallery:** View all your images and videos in a single unified gallery with intelligent caching and skeleton loading.
- **Secure Management:** Manage your data through one clean dashboard with complete privacy. Your files stay on the cloud providers.

## ✨ Key Features

**🔐 Secure Authentication**
- OTP (One-Time Password) email login
- Social login support via Google OAuth
- Protected, stateless JWT user sessions

**☁️ Multi-Cloud Connectivity**
- Link multiple cloud accounts (Google Drive) simultaneously
- Seamless access to distributed storage
- Real-time tracking of storage limits and used space across all accounts

**🖼️ Unified Media Gallery**
- Aggregates images and videos from all connected platforms
- High-performance, infinite-scrolling grid
- Intelligent backend caching to preserve Google API rate limits

**📤 Smart Upload Hub**
- Premium drag-and-drop file upload interface
- Disk-streaming backend architecture to effortlessly handle large files (up to 5GB) without crashing the server
- **Auto-Routing:** Automatically detects the cloud account with the most available storage and uploads the file there.

---

## 🔄 How It Works

1. Create an account and log in securely via OTP.
2. Connect your cloud services (granting Google Drive read/write permissions via OAuth).
3. Unicloud securely queries your cloud storage. Your data is never stored on our servers.
4. Your files and photos appear instantly in one unified interface.
5. Drag and drop a file into the Upload Hub and watch it intelligently route to the optimal cloud account.

---

## 🎯 Problem It Solves

Managing multiple cloud accounts is fragmented and time-consuming. Users often store files across different platforms but lack a single view to access them efficiently. Moving a massive file requires checking each account's remaining quota one by one.

Unicloud removes this friction by acting as a central intelligence layer over all cloud services, giving users a unified and radically simplified experience.
